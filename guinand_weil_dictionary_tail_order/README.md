# Zero-Source Transport Reproducibility Package

This directory contains the Paper 2 manuscript and the companion verification
package for

```text
A finite Guinand-Weil dictionary and archimedean tail order
for the truncated Weil quadratic form
```

Akiva Groskin, 2026. Preprint: [arXiv:2607.02828](https://arxiv.org/abs/2607.02828)
(math.NT, math.SP). Archived on Zenodo, concept DOI
[10.5281/zenodo.21124802](https://doi.org/10.5281/zenodo.21124802) (always
resolves to the latest version).

The paper is a compact theorem paper (12 content pages, two figures, one
table) with two theorem spines and three corollaries:

1. the exact finite Guinand-Weil zero-source dictionary (Theorem 2.5), with an
   entry-identification lemma pinning the source assembly to the CCM closed
   forms at equation level (Lemma 2.1);
2. the exact finite source quotient of dimension `2N+1`, stated as an iff
   (Corollary 2.4);
3. the positive-dimensional non-collapsing pole-neutral finite subfamily
   (Corollary 2.7);
4. the finite-T archimedean tail-order theorem with strict observability and
   strict total positivity of the post-band tail increment (Theorem 3.2);
5. a two-sided finite certification rule with an explicit closed-form budget
   and the asymptotic `B_T = (2N+1) rho (log(T/2pi)+1)/(pi^2 T)(1+o(1))`
   (Corollary 3.3), plus a self-contained envelope lemma for `h_+`
   (Lemma 3.1).

The manuscript includes a worked example verified against the first 512
nontrivial zeros of zeta (Table 1, Figure 1) and an eigenvalue-flow
demonstration of the certification rule (Figure 2).

## Files

Core manuscript:

- `main.tex`, `main.bib`, `main.bbl`, `main.pdf`: manuscript and compiled paper.
- `fig_dictionary.pdf`, `fig_tailorder.pdf`: figures (generated by
  `make_figures.py`).

Core reproducibility scripts:

- `audit_exact_series_identity.py`: exact integer-arithmetic audit for the
  single-frequency identity.
- `audit_kernel_span_rank.py`: exact audit for the finite Volterra-kernel
  span theorem.
- `audit_full_matrix_source_quotient.py`: exact audit that full finite
  divided-difference source matrices factor through the `2N+1` source
  quotient.
- `audit_pole_neutral_survival.py`: exact symbolic audit for the pole-square
  factorization, moment-row independence, and (as exact rank computations over
  `Q(b)`) the dimension formula `dim = N-s-1`.
- `verify_finite_dictionary.py`: guard for the single-frequency source
  identity and the pole normalization, the latter by three independent routes
  (y-integral, two-sided xi-integral, closed-form square).
- `verify_dictionary_threeroute.py`: the three-route confirmation of the
  dictionary theorem (closed-form CCM assembly vs source-side evaluation vs
  zero-side sums over the first 512 zeros), at a generic vector, a
  pole-neutral vector, and the worked-example vector.  Also generates the
  reference constant used by `verify_zero_side.py`.
- `verify_arch_tail_order.py`: guard for the finite-T archimedean tail-order
  algebra, certified `h_+` threshold, and strict-total-positivity smoke cases.
- `audit_arch_tail_dt_bridge.py`: three-route guard that the rank-two Cauchy
  tail density is the finite-T derivative of the archimedean source term.
- `arch_tail_budget.py`: Arb interval tail-budget script for the `c=100`,
  `N=200`, `T=800` envelope.
- `arch_tail_stress_ladder.py`: deterministic interval stress ladder for the
  tail bounds across `T`, `N`, and Arb precision.
- `arch_tail_exact_asymptotic.py`: exact `B_T` versus the closed asymptotic of
  Corollary 3.3(iii); the `h_+'` envelope ladder; the `B_T = 1e-59` solve.
- `arb_ldlt_certify.py`: rigorous Arb interval `LDL^T` inertia certificate of
  the cutoff-free block (generator of the 9000-bit `c=100, N=200`
  certificate), with a self-test validating the Arb entry balls against an
  independent mpmath recomputation.
- `verify_zero_side.py`: the original `M <= 64` zero-side confirmation for the
  `c=13, N=4` regression vector (superseded in depth by
  `verify_dictionary_threeroute.py`; retained for continuity).
- `make_figures.py`: generates both manuscript figures from package artifacts.

mpmath suffices for everything except the three Arb interval scripts
(`verify_arch_tail_order.py`, `arch_tail_budget.py`,
`arch_tail_stress_ladder.py`, `arb_ldlt_certify.py`), which need python-flint,
and `make_figures.py`, which needs matplotlib.

## Build The Paper

```bash
pdflatex -interaction=nonstopmode -halt-on-error main.tex
bibtex main
pdflatex -interaction=nonstopmode -halt-on-error main.tex
pdflatex -interaction=nonstopmode -halt-on-error main.tex
```

## Run The Core Gate

```bash
PYTHONDONTWRITEBYTECODE=1 python3 audit_exact_series_identity.py
PYTHONDONTWRITEBYTECODE=1 python3 audit_kernel_span_rank.py
PYTHONDONTWRITEBYTECODE=1 python3 audit_full_matrix_source_quotient.py
PYTHONDONTWRITEBYTECODE=1 python3 audit_pole_neutral_survival.py
PYTHONDONTWRITEBYTECODE=1 python3 verify_finite_dictionary.py
PYTHONDONTWRITEBYTECODE=1 python3 audit_arch_tail_dt_bridge.py
PYTHONDONTWRITEBYTECODE=1 python3 verify_zero_side.py
PYTHONDONTWRITEBYTECODE=1 python3 verify_dictionary_threeroute.py 512
PYTHONDONTWRITEBYTECODE=1 python3 arch_tail_exact_asymptotic.py
# with python-flint:
PYTHONDONTWRITEBYTECODE=1 python3 verify_arch_tail_order.py
PYTHONDONTWRITEBYTECODE=1 python3 arch_tail_budget.py
PYTHONDONTWRITEBYTECODE=1 python3 arch_tail_stress_ladder.py
PYTHONDONTWRITEBYTECODE=1 python3 arb_ldlt_certify.py --selftest --c 13 --N 8 --prec 300
```

The headline 9000-bit certificate is regenerated (about 15 minutes on an
Apple M2 Max core) by

```bash
python3 arb_ldlt_certify.py --selftest --c 100 --N 200 --prec 9000 \
        --json-out c100_N200_arb_ldlt_prec9000_provenance.json
```

Release checksums are recorded in `SHA256SUMS`.
